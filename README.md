# ☕ Silva Café Website

Welcome to **Silva Café**, a responsive website built with **HTML** and **CSS**. It showcases a fictional coffee shop with a modern, cozy, and welcoming atmosphere.

This project was developed as a front-end practice exercise, with a focus on **responsive web design**, **clean UI**, and **German-language content**.

---

## 🌐 Live Demo

👉 https://thiagosilvaqa93.github.io/Silva-s-Cafeteria/

---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Smooth navigation between sections
* Coffee menu with images and promotional prices
* Customer testimonials section
* Embedded Google Maps location
* Social media icons and interactive elements

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Google Fonts** (Roboto)
* **Icons8** (SVG icons)
* **Google Maps Embed API**

---

## 🚀 Running the Project Locally

1. Clone the repository:

```bash
git clone https://github.com/thiagosilvaqa93/silva-cafe.git
```

2. Navigate to the project folder:

```bash
cd silva-cafe
```

3. Open `index.html` in your preferred web browser.

You can either:

* Double-click the `index.html` file.
* Use **Live Server** in Visual Studio Code for a better development experience.
